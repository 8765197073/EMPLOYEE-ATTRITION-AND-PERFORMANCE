# Employee Attrition and Performance Analysis

## 🚀 Project Overview
A comprehensive data analytics project aimed at identifying key drivers behind employee attrition and performance metrics. This project leverages data-driven insights to help HR departments optimize talent retention strategies and improve overall workforce management.

## 🛠 Tech Stack
* **Language:** Python
* **Data Analysis/Manipulation:** Pandas, NumPy
* **Visualization:** Power BI / Matplotlib / Seaborn
* **Environment:** Jupyter Notebook

## 📈 Key Insights
* **Attrition Patterns:** Identification of demographic and professional factors contributing to turnover.
* **Performance Correlations:** Analysis of the relationship between tenure, department, and performance ratings.
* **Predictive Modeling:** Insights derived to help proactively manage at-risk employees.

## 📂 Project Structure
* `/data` - Contains the raw and processed datasets.
* `/notebooks` - Jupyter notebooks containing the step-by-step EDA and modeling.
* `/dashboard` - Visual summaries and reports.
